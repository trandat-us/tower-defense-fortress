# Tower Defense: Fortress - Godot Engine

A 3D tower defense game developed with Godot. This project was created to practice 3D top-down gameplay design and programming.

Players can freely move around each level, observe enemy paths, and build strategies by placing, upgrading, or demolishing towers. Defeat enemies to earn gold and defend against incoming waves.

## 🚀 Features
- Free camera movement to explore each level and plan strategies
- Wave-based enemy spawning with unique attributes and behaviors
- Five levels with distinct maps, enemy paths, and environments
- Three tower types with unique stats, mechanics, and upgrade paths
- Build, upgrade, and demolish towers dynamically
- Earn gold by defeating enemies
- Speed boost feature to accelerate gameplay
- Clean and intuitive UI/HUD
- Basic player progression system (all levels currently unlocked)

## 🎮 Gameplay
- Move camera: WASD  
- Place tower: Drag & drop from tower cards (Left Mouse Button)  
- Select tower: Left Mouse Button  

## ⚙️ Installation
- Use Godot version 4.6  
- Clone the repository and unzip it  
- Import the **project.godot** file into Godot Hub  
- Press F5 to run
